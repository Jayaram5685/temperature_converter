# 🌡️ Temperature Converter 🔥

Welcome to the **Temperature Converter** project! This simple web application helps you convert temperatures between **Celsius**, **Fahrenheit**, and **Kelvin** effortlessly. Whether you're cooking, traveling, or working with scientific data, this tool will make temperature conversion a breeze. 

## 🚀 Features

- **Celsius ↔ Fahrenheit**: Easily convert between Celsius and Fahrenheit.
- **Celsius ↔ Kelvin**: Convert Celsius to Kelvin and vice versa.
- **Fahrenheit ↔ Kelvin**: Switch between Fahrenheit and Kelvin quickly.
- **Interactive & Real-time**: As soon as you input the value, the conversion happens instantly!

## 💻 Technologies Used

- **HTML5**: Structure of the page.
- **CSS3**: Styling to make the app look appealing.
- **JavaScript**: Logic to handle the conversion and interaction.

## 🔧 How to Run Locally

To get started with this project on your local machine:

1. Clone the repository:

    ```bash
    git clone https://github.com/Jayaram5685/temperature_converter.git
    ```

2. Navigate to the folder:

    ```bash
    cd temperature_converter
    ```

3. Open the `index.html` file in your browser to use the temperature converter.

## ✨ How it Works

1. **Select the Unit**: Choose the unit (Celsius, Fahrenheit, or Kelvin) for the temperature you want to convert.
2. **Input Temperature**: Enter the temperature value in the input box.
3. **Instant Conversion**: As you type, the converted values will be displayed instantly for all other units.
